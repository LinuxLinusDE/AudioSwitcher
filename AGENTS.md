# AGENTS

## Erledigt

- Option `--force-shuffle-audio-input` hinzugefuegt.
- `--force-shuffle-audio-input` erzeugt immer eine neue zufaellig gemischte MP3 aus `audio-input/`, auch wenn in `audio/` schon eine MP3 vorhanden ist.
- `--video-input` unterstuetzt jetzt mehrere Dateien, Ordner und Glob-Pattern wie `"/pfad/zu/*.mp4"`.
- `--force-shuffle-audio-input` erzeugt bei mehreren Videos pro Video eine eigene Tracklist passend zum Video-Output.
- Warnung, wenn die MP3s aus `audio-input/` kuerzer als das Zielvideo sind.
- CLI-Assistent fuer Aufrufe ohne Argumente hinzugefuegt.
- Assistent fragt den Audio-Input-Ordner ab.
- Assistent nutzt eine klare Audio-Modusauswahl und zeigt vor dem Start eine Preflight-Uebersicht.
- Tracklists enthalten Videonamen, Videolaenge und Audiolaenge.
- MP3-Gesamtlaenge wird vor der Verarbeitung angezeigt.
- Option `--output-dir` fuer einen separaten Zielordner hinzugefuegt.
- README-Dateien um die neuen Optionen ergaenzt.
